# Code of Conduct

## Our standard

We are committed to a friendly, safe, and welcoming environment for everyone.
We expect participants to be respectful, assume good faith, accept constructive
feedback gracefully, and focus on what is best for the project and community.

Unacceptable behavior includes harassment, personal attacks, discriminatory
language, and publishing others' private information without consent.

## Scope

This applies within all project spaces — issues, pull requests, discussions —
and in public spaces when an individual represents the project.

## Enforcement

Report unacceptable behavior privately to the repository owner. Maintainers may
remove, edit, or reject contributions that violate this code, and may ban
temporarily or permanently for behavior they deem inappropriate.

This Code of Conduct is adapted in spirit from the Contributor Covenant.
